# Bank Database SQL

A simple banking database management system built with **MySQL** as part of a **Data Management (DTS304)** course project at the **Federal University of Technology, Minna (FUTMINNA)**.

The project demonstrates the fundamentals of relational database design by modeling a real-world banking system and implementing it using SQL.

## Features

- Relational database schema
- Normalized table design
- Primary and foreign key constraints
- Sample data for testing
- SQL queries for data retrieval and analysis
- ER Diagrams (Chen's Notation & Crow's Foot)

## Database Entities

- Customer
- Branch
- Account
- Card
- Loan
- Employee
- Transactions

## Technologies

- MySQL 8.0
- MySQL Workbench

## Learning Objectives

This project was created to practice:

- Database design
- Entity-Relationship (ER) modeling
- SQL Data Definition Language (DDL)
- SQL Data Manipulation Language (DML)
- SQL queries and joins
- Foreign key relationships and referential integrity
